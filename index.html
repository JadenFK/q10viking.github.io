Eclipse快捷键

发表于 2018-09-20 | 分类于 SSM学习

XML文件

  • ctrl+shift+/ 注释
  • ctrl+shift+\ 取消注释
  • Alt+/ 自动补全

Java code

  • 定位在变量上，然后ctrl+1，选择Convert local variable to field——快速声明为成员变量
  • 选中目标Alt+上下箭头，进行移动
  • Alt+shift+Z 将代码包围起来，如if,try等
  • 抽取局部变量，Alt+shift+L
  • Ctrl+PageUp/PageDown切换代码页面

关联源码

  • 按住ctrl键鼠标放到代码上，然后点击Open Declaration，然后点击Attach Source,选择源码zip文件

  • Alt+<– 源码回到上一步

Eclipse问题

  • 【问题1：】项目出现红色感叹号

    比如由于之前导入的mysql-connector-java-5.1.7-bin.jar在build path之后，又删除了该jar包，换成新的mysql-connector-java-8.0.12.jar包，然后build path.
    此时eclipse的reference library中仍然残存着mysql-connector-java-5.1.7-bin.jar，我们需要点击项目—>build path—>Configure build path,将libraries中以前的mysql-connector-java-5.1.7-bin.jar信息删除

复制项目

  • 直接ctrl+C复制然后Ctrl+v粘贴，改一个不同的名字即可
