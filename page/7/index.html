os

  • os.getcwd()

    • return a string representing the current working directory

  • os.path.exists(path)

    • Return true if path refers to an exsiting path.Or return false.

multiprocessing

  • Pool 的map方法，开启进程

haslib

  • MD5 hash
    This hash function accepts sequence of bytes and returns 128 bit hash value,usually used to check data intergrity.
    Function associated
    • encode(): converts the String into bytes to be acceptable by hash function
    • digest(): Returns the encoded data in byte format
    • hexdigest: Returns the encoded data in hexadecimal format

  • 打开文件

  • 打开进程池
