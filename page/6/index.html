Eclipse快捷键

发表于 2018-09-20 | 分类于 SSM学习

XML文件

  • ctrl+shift+/ 注释
  • ctrl+shift+\ 取消注释
  • Alt+/ 自动补全

Java code

  • 定位在变量上，然后ctrl+1，选择Convert local variable to field——快速声明为成员变量
  • 选中目标Alt+上下箭头，进行移动
  • Alt+shift+Z 将代码包围起来，如if,try等
  • 抽取局部变量，Alt+shift+L
  • 抽取局部方法，Alt+shift+M
  • Ctrl+PageUp/PageDown切换代码页面
  • 输入sysout 然后快捷键Alt+/

生成Java-Beans的套餐

在beans中只要写出了成员属性，就可以使用快捷键快速生成setter,getter和toString方法

  • Shift+Alt+S 快速列出清单
  • 然后根据需要，用ctrl+下划线的字母进行选择
    • 如： 生成setter,getter方法ctrl+r
    • 如： 生成toString方法，ctrl+s

关联源码

  • 按住ctrl键鼠标放到代码上，然后点击Open Declaration，然后点击Attach Source,选择源码zip文件

  • Alt+<– 源码回到上一步

  • open type窗口查看想看的类： ctrl+shift+t

类相关

  • 查看一个类或者接口的继承实现关系： 选中该类名，按ctrl+T
  • 查看一个类或者接口的结构： 选中该类名，按ctrl+O

Eclipse视图

项目视图
  • 【项目栏太多，编程时只想关注一个】
  • 解决办法： 在左侧项目导航栏标题栏中，右上角有一个向下的三角，选择select working set ，会打开工作视图设置界面，新建一个工作视图，点new按钮，然后选择你常用的工程，就可以。日常工作的时候就可以点那个向下的三角，来选择你想要的工作视图。

Eclipse问题

  • 【问题1：】项目出现红色感叹号

    比如由于之前导入的mysql-connector-java-5.1.7-bin.jar在build path之后，又删除了该jar包，换成新的mysql-connector-java-8.0.12.jar包，然后build path.
    此时eclipse的reference library中仍然残存着mysql-connector-java-5.1.7-bin.jar，我们需要点击项目—>build path—>Configure build path,将libraries中以前的mysql-connector-java-5.1.7-bin.jar信息删除

  • 【问题2：Eclipse MarketPlace点击无反应】

    • Eclipse版本Photon Release (4.8.0)，点击help->Eclipse MarketPlace无任何反应

  • 解决办法： help->install new software输入

    重新安装marketPlace即可

复制项目

  • 直接ctrl+C复制然后Ctrl+v粘贴，改一个不同的名字即可

Eclipse扩展功能

  • Eclipse中Spring配置文件Class属性中全路径自动提示配置
