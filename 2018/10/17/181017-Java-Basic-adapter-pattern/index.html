适配器模式(Adapter Pattern)

发表于 2018-10-17 | 分类于 Java基础

笔记本电脑充电器的那个黑盒子就相当于适配器。一般在中国能用，在日本也能用，虽然两个国家的电源电压不同，中国是220V，日本是110V,但是这个适配器都会把这些不同的电压转换为电脑需要的36V电压（我没有考证），保证你的笔记本能够正常运行。

Intent

  • Convert the interface of a class into another interface the clients expect.
  • Adapter lets classes work together that couldn’t otherwise because of incompatible interfaces.（wrap an otherwise incompatible object in an adapter to make it compatible with another class.）

通用类图

  • Target是一个类（接口）
  • Adaptee是一个接口，通过Adapter把Adaptee包装成Target的一个子类（实现类）

Applicability

Use the Adapter pattern when:

  • you want to use an existing class,and its interface does not match the one you need.
  • you want to create a reusable class that cooperates with unrelated or unforeseen classes,that is ,classes that don’t necessarily have compatible interfaces.
  • you need to use several existing subclasses,but it’s impractical to adapt their interface by subclassing every one.An object adapter can adapt the interface of its parent class.
  • most of the applications using third party libraries use adapters as a middle layer between the application and 3rd party library to decouple the applicatioin from the library.If another library has to be used only an adapter for the new library is required without having to change the application code.
