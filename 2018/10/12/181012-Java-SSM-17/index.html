Spring与DAO(JdbcTemplate)

发表于 2018-10-12 | 分类于 SSM学习

  • Github-相关代码

  • Spring与DAO部分是Spring两大核心技术Ioc与AOP的典型应用体现

  • 对于JDBC模板使用，是IoC的应用，将JDBC模板对象注入给了Dao层的实现类
  • 对于Spring的事务管理，是AOP的应用，将事务作为切面织入到了Service层的业务方法中。

JDBC模板

  • Spring提供了一个强有力的模板类-JdbcTemplate来简化JDBC操作

  • 数据源DataSource对象与模板JdbcTemplate对象均可以通过Bean的形式定义在配置文件中，充分发挥依赖注入的威力

  • 添加spring-jdbc-jar包还要添加spring-tx-jar包，一起添加到项目中。

  • DAO的实现类需要继承JdbcDaoSupport

    • 有JdbcTemplate属性，用户接收JDBC模板
    • 需要自己设置JdbcTemplate的值

    • JdbcDaoSupport.java中的源码

    • JdbcTemplate继承自JdbcAccessor，Accessor中有一个DataSource属性

  • 在JdbcDaoSupport类中，发现其有一个dataSource属性，查看setDataSource()方法体可知，若JDBC模板为null,则会自动创建一个模板对象

所以配置文件两种方式

  • 通过注册JdbTemplate

  • 直接使用数据源

DateSource数据源的配置

Spring默认的数据源DriverManagerDataSource

数据库连接四要素，驱动，url,用户名,密码

  • Spring默认的数据源为DriverManagerDataSource,其中有一个属性DriverClassName,用于接收DB驱动
  • DriverManagerDataSource继承了AbstractDriverBasedDataSource，这个类中包含了，url,username,password等属性
DBCP数据源BasicDataSource
  • DBCP,DataBase Connection Pool，是apache下的项目，使用该数据源需要导入两个Jar包
    • dbcp与pool
C3P0数据源 ComboPooledDataSource
  • C3P0数据源是ComboPooledDataSource,Ctrl+O查看其类结构可看到，其有driverClass,jadbcUrl,username,password

从属性文件读取数据库连接信息

  • Spring配置文件从属性文件中读取数据时，需要在< propertys / >的value属性中使用${ },
方式-使用 class 为 PropertyPlaceholderConfigurer
  • location属性指定文件位置

这种方法不常用

context:property-placeholder/方式
  • 需要添加context的约束。
  • < context:property-placeholder / > 标签有一个location属性，用于指定文件位置

注意properties文件中: 最好使用jdbc.username两级结构而不是username一级结构。

对DB增、删、改操作

  • 对DB的增，删，改都是通过update()方法实现的。

对DB的查询操作

JDBC模板的查询结果都是以对象的形式返回的。根据返回对象类型的不同，分为

  • 简单对象查询： 查询结果为String,Integer等简单对象，或该类型作为元素的集合类型，如List< Integer >
  • 自定义对象查询： 查询结果为自定类型，如User等，或者该类型作为元素的集合类型，如List< User >
简单对象查询
  • 查询结果为单个对象的queryForObject()与查询结果为List的queryForList()
自定义对象查询

  • 查询结果为单个对象的queryForObject()与查询结果为List的query()

  • 使用RowMapper接口对数据进行封装

  • 参数rowNum表示总的结果集中当前行的行号，但是参数代表的是当前行的结果

JdbcTemplate对象是多例的

  • 系统会为每一个模板对象的线程（方法）创建一个JdbcTemplate实例，并且在该线程（方法）结束时，自动释放JdbcTemplate实例。
  • 所以在每次使用JdbcTemplate对象时，都需要通过getJdbcTemplate（）方法获取。
