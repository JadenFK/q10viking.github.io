MyBatis注解式开发

  • Mybatis的注解，主要是用于替换映射文件
  • 要想发挥mybatis功能，还是要使用映射文件，不建议使用动态注解来开发

注解的基础语法

  • 注解后没有分号
  • 注解首字母是大写的，因为注解与类，接口是同一级别的。一个注解，后台对应着一个@interface类
