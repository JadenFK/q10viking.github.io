Hexo提交失败

发表于 2018-04-18 | 分类于 hexo

problem

Reason
Github isn’t able to authenticate you.So,you aren’t setup with an SSH key.

Solution

  • Step-1:将hexo与github page 联系起来

  • Step-2: 输入 cd ~/.ssh,检查是否有.ssh文件夹

  • Step-3: 输入ls,列出该文件下的内容，下图说明存在

  • Step-4: 输入 ssh-keygen -t rsa -C “1403090523@cau.edu.cn“,连续三个回车，生成密钥，最后得到两个文件: id_rsa和id_rsa.pub.(我的路径是：C:\Users\Viking.ssh)

  • Step-5: 输入eval “$(ssh-agent -s)”,添加密钥到ssh-agent

  • Step-6: 输入 ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent

  • Step-7: 登录Github,点击头像下的settings,添加ssh

  • Step-8: 新建一个new ssh key,将id_rsa.pub文件里的内容复制上去

  • Step-9: 输入 ssh -T git@github.com，测试添加ssh是否成功，如果看到Hi后面是你的用户名，就说明成功了。
