Reader家族

Reader与Writer的出现主要是解决编码，直接面对unicode字符，而InputStream面对是的是字节。

InputStream与Reader区别

在字节流中，涉及到编码与解码的问题，传输双方必须用同一种编码方式才能正确接收。
传输方： 1）将需要传输的字符编码成指定字节（本次是保存文件时采用的编码方式） 2）传输字节

接收方： 1） 接收字节 2）将字节解码成对应的字符。（相同的编码方式）
