java-8(2) Putting lambdas into practice

发表于 2018-09-04 | 分类于 Java基础

Lambda can be used in practice to make your code more flexible and concise.

The execute around pattern

Execute around pattern is one of the Design Patterns.

A recurrent pattern in resource processing(for example,dealing with files or databases) is to open a resource,do some processing on it,and then close the resource.
The setup and cleanup phase are always similar and surround the important code doing the processing.This is called execute around pattern（环绕模式）

$\blacksquare$ Step1: we deal with a file resource

  • The current code is limited.You can read only the first line of the file.What if you’d like to return the first two lines instead or even the word used most frequently?
  • Ideally,you’d like to reuse the code doing setup and tell the processFile method to perform different actions on the file
  • You need a way to pass behavior to processFile method so it can execute different behaviors uing a BufferedReader.

Improved by-Behavior parameterization

Using JavaAPI functional interfaces

  • Functional interfaces are useful because the signature of the abstract method can describe the signature of a lambda expression.
  • So in order to use different lambda expressions,you need a set of functional interfaces that can descibe common function descriptors.
  • The Java library designers for java 8 have helped you by introducing several new functional interfaces inside the $java.util.function$ package.

$\blacksquare$ Predicate

java.util.function.Predicate< T >

You might want to use this interface when you need to represent a boolean expression that uses an object of type T.
PredicateMain.java illustrate using Predicate interface to filter non-empty string.

$\blacksquare$ Consumer

java.util.function.Consumer

You might use this interface when you need to access an object of type T and perform some operations on it.

For example,you can use it to create a method forEach,which takes a list of Integers and applies an operation (such as: print the element) on each element of that list.

$\blacksquare$ Function

java.util.function.Function

You might use this interface when you need to define a lambda that maps information from an input object to an output.(such as mapping string to its length)

Example: mapping strings to its length

Keep in mind: These functional interfaces supplied by Java 8 API are only a starter kit.You can always make your own if needed!
