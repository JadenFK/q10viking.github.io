爬虫转义字符问题

发表于 2018-09-13 | 分类于 python scraping
  • python会自动把反斜杠\加上转义
  • 比如爬取的信息中含有\单斜杠，会变成对应的双反斜杠\.即: 把单斜杠个转义了。
  • 这样在处理json字符串形式的数据时，要想转化成json对象，会出现问题

简单问题

感谢这篇文章【python爬取html，反转义，双斜杠(\)转换成单斜杠()字符被json正常解析】

【问题】

【解决方法一】

  • 使用re.sub方法

  • 注意，re.sub方法中，模式加上r前缀，否则会报错

【解决方法二】

  • eval（）函数用来执行一个字符串表达式，并返回表达式的值

  • 补充eval函数用法

$\blacksquare$ 【注意】

  • 最好使用eval来消除转义的困扰，因为使用re.sub会将中文的unicode编码\ \u5408 给替换成u5408，而我们需要的是\u5408,这样的形式
