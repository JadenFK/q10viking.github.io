分析Ajax抓取今日头条美图

发表于 2018-09-13 | 分类于 python scraping

目标站点分析

  • 抓取目标是组图（是分页显示的，而不是在一张网页上呈现）如早安，秋天


  • 查看网页的源码，发现都是javascript代码，并没有我们想要的数据。（这是因为该网页采用了Ajax技术，整个网页呈现的数据是通过Ajax后台返回的json数据，渲染而来的
  • 接下来查看ajax到底请求了什么内容，该内容是否有我们需要的数据。从All切换到XHR,也就是Xml HttpRequest的缩写。发现返回的是json数据；
  • 我们点开data，查看第0条数据，我们发现其中有条数据【title=”写给秋天……”】和我们在网页上的对应，同时发现还有【source_url=”/group/6589058786369995272/“】,点击网页上的写给秋天…,发现后面的数字部分都一样。由此我们知道该ajax发送的请求返回了我们需要的数据。
  • 点击网页上的【图集】，切换到我们要抓取的页面，做同样的，分析
  • 在网页上我们一直往下拉，发现ajax一直在向后台发送请求，返回回来我们需要的数据，我们观察到下来时的【索引页offset】的变化

  • 接下来我们分析详情的网页，但进去【早安，秋天】页面
  • 我们在XHR还是在JS中都没有发现我们想要的数据，经过仔细的查询，我们发现在原生的Doc中有图片的请求链接数据，它隐藏在javascript 代码中

  • $\blacksquare$ 【1】抓取索引页内容
    • 利用requests请求目标站点，得到索引网页的HTML代码，分析出里面我们想要的详情页的url。
  • $\blacksquare$ 【2】 抓取详情页
    • 解析详情页返回结果，在请求也中抓取我们所需要的信息，如每张图片的链接，该网页的title等

  • $\blacksquare$ 【3】下载图片与保存数据库

    • 将图片下载到本地，并把页面信息及图片URL保存至MongDB

  • $\blacksquare$ 【4】开启多进程加速爬取

    • 对多页面内容进行遍历，开启多进程提高抓取速度
