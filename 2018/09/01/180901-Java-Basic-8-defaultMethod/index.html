Java-8 Default Method

发表于 2018-09-01 | 分类于 Java基础

What default methods are

  • Traditionally,a Java interface groups related methods together into a contract.

  • Any class that implements an interface must provide an implementation for each method defined by the interface.

  • This cause a problem when library designers need to update an interface to add a new method,the existing concrete classes need to be modified to reflect the new interface contract.

  • Java 8 API introduces many new methods on existing interfaces,such as the sort method on the List interface.(Cause the problem,the existing collection frameworks such as GUAVA and Apache Commons who need to modify all the classes implementing the List interface to provide an implementation for the sort method too. )

But don’t worry.Java 8 introduces a new mechanism to tackle this problem.

  • Java 8 allow static methods inside interfaces.
  • Java 8 introduces a new feature called default methods that allows you to provide a default implementation for methods in an interface.

As a result,existing classes implementing an interface will automatically inherit the default implementation if they don’t provide one explicitly
