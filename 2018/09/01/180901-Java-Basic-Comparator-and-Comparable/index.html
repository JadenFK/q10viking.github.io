Comparable VS Comparator

发表于 2018-09-01 | 分类于 Java基础

Comparable interface

If any class implements Comparable interface then collection of that object can be sorted automatically using Collection.sort() or Arrays.sort().Object will be sort on the basic of compareTo method in that class.

Comparable interface

Example
Country object implements Comparable interface,then use Collections.sort() to sort the collection of Country.

Comparator interface

When you want to sort the list of objects of a class,you can use Comparator interface.

$\blacksquare$ You don’t need to implement Comparator on the class whose objects need to be sorted.

$\blacksquare$ You can create a separate class and implement a Comparator interface（use different sorting logic based on different attributes of object that needs to be sorted）.

Example

You want to sort list of employees by ID,you can use below code to do that.

Create a class named EmployeeSortByIdComparator.java.This class will have logic to sort list of Employee by empID.(However,instead of it we can use lambda in java 8,to simple our code,make it more readable.)

Different Comparator VS Comparable

Parameter Comparable Comparator
Sorting Logic Sorting logic must be in same class whose object are being sorted.Hence this is called natural ordering of objects Sorting logic is in separate class.Hence we can write different sorting base on different attributes of objects to be sorted.E.g Sorting by id,name etc
package java.lang.Comparable java.util.Comparator
Sorting Method int CompareTo(Object o1) This method compares this object with o1 object and returns a integer.Its values has following meaning. $\blacksquare$ positive - this object is greater than o1 . $\blacksquare$ zero - this object equals to o1. $\blacksquare$ negative - this object is less than o1 int compare(Object o1,Object o2) This method compare o1 and o2 objects and return a integer.Its value has following meaning. $\blacksquare$ positive - o1 greater than o2 . $\blacksquare$ zero - o1 equals to o2. $\blacksquare$ negative - o1 is less than o2
Calling method Collections.sort(List) Collections.sort(List,Comparator) or List.sort(Comparator)

Example
You can sort a list of Strings since the String class implements Comparable,and the String.compareTo compares strings in dictionary order

Now suppose we want to sort strings by increasing length,not in dictionary order.The String class isn’t ours to modify.So to deal with this problem,we need Comparator interface.
