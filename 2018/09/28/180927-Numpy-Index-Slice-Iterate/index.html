补充

  • Notice how the shape attribute affects broadcasting along an axis.

  • if we broadcasting along the zero axis,such as a.sum(0),we eliminate the zero element in the shape attribute,The result is(3,5)(dim0 去掉产生的结果是3x5)

  • If we broadcasting along the one axis,we eliminiate the one element ,and the result is (2,5)
  • So,of course a.sum(2),the result is (2,3)
