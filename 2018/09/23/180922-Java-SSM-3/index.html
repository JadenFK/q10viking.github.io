SQL预编译问题

  • 当Java代码通过JDBC的Statement向DB发送一条SQL语句时，DBMS会对SQL语句编译后执行

  • 当Java代码通过JDBC的PreparedStatement向DB发送一条SQL语句时

    • DBMS会首先编译SQL语句
    • 然后将编译好的SQL语句放入到DBMS的数据库缓存池中再执行。
    • 当DBMS再次接收到对该数据库操作的SQL语句时，先从DB缓存池中查找该语句是否被编译过，若被编译过，则直接执行，否则先编译后将编译结果放入DB缓存池，再执行。(这性能得到提升)
