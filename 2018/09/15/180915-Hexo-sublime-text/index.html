Sublime Text

发表于 2018-09-15 | 分类于 hexo

【Sublime Text 3 Download】

  • 目前是为了方便编辑python代码
  • 所以这篇文章的核心是配置python环境

恢复默认设置或彻底卸载干净

  • 【问题：】卸载Sublime Text 3，再次重新安装，发现上一次安装的配置文件仍然有效并没有恢复到默认的设置
    • 【解决办法：】在菜单栏选择Preferences–>Browse Packages，在打开的目录中将文件全部删除即可。（因为此处存着用户数据，卸载或者升级时是不会删除）

package control(installed package)

通过添加第三方包来增加sublime text的功能

  • Tools—>control palette,输入： install package control

$\blacksquare$ 配置颜色和主题

  • 安装predown主题，在输入框中输入： predown,然后安装它，观察到sublime text的最下面会显示正在安装的信息提示

  • 同样的安装material theme

  • 设置user-setting,在preferences—>setting中,打开后左边是default setting,右边是user-setting,我们需要更改配置的是右边的。打来【配置文件链接】，将里面的内容粘贴到user-default中，然后保存。

  • 【sublime text文档中说明每个属性的作用】并且sublime text中default setting 中的注释也有说明

  • 安装【bracketHighlighter】,能够使得开始的括号与结束括号高亮，使得编程时，能够知道这段代码是属于哪个部分

  • 【sideBarEnhancements】增加对左边文件的操作功能选项，如：copy,reflesh等

Anaconda

Anaconda turns your Sublime Text 3 into a full featured Python IDE.链接

  • 安装【Anaconda】，这个Anaconda只是sublime text的package，不是Anaconda软件，他们只是名字相同而已。
    • 配置Anaconda设置响应的属性，preferences—>Package Settings—>Anaconda—>settings-User,同样的每个属性的含义也可以在preferences—>Package Settings—>Anaconda—>settings-Default中看见。

自动代码的补全效果，在配置文件settings-user已经配置

  • 小技巧: 找anacoda的路径(前提是使用的python是属于anaconda的)

补充

我们看到了有两种不同的setting

  • 一种是关于sublime text的setting,如使用什么主题，皮肤，字体大小，滚动的速度等（Preferences—>setttings）
  • 一种是关于对插件的设置，如Anaconda插件，（Preferences—>Package settings—>Anaconda—>Settings-User）
package control(uninstall package)
  • 如果是从package control 安装的package,可以在Preferences—>Package Control 中输入 remove package,选择Package control: Remove Package，然后在弹出的列表中选择想要删除的信息

console中文乱码问题

  • 安装ConvertToUTF8
  • 重启选择File–>Set File Encoding to—>Chinese Simplified(GBK)
  • 默认的编译器可以直接按Ctrl+B编译运行Python，并在控制台输出结果.美中不足的是无法输出中文，需要自己手动配置一番

  • 在Sublime Text 3 中依次点击菜单Tools->build system->New build system，粘贴下方的代码并保存为Python_CN.sublime-build。

  • 重启Sublime,选择Tools->build system->Python_CN即可

SublimeREPL的强大

在sublime text中安装SublimeREPL

  1. 支持输入
  2. 支持调试
  • 步骤：Preferences–>Browse Packages–>SublimeREPL文件夹–>config文件夹–>Python文件夹–>Default.sublime-commands(以文本格式打开)

这是sublimeREPL的配置文件，找到你需要的命令复制下来
然后粘贴到： Preferences–>Key Bindings User

小技巧

  • 【1: 】将文件夹拖拽到sublime Text中能够快速打开此文件夹目录以及浏览其中的文件。
  • 【2: 】退出control palette 使用Esc键
  • 【3: 】编译运行python程序，Tools—>build System勾选上python，然后Tools—>build 运行程序，有错误发生时，会自动在错误的行提示错误信息(这很赞)
  • 【4: 】查看已经安装的package,preferences—>package control—>list packages
  • 【5: 】 分屏显示，View—>Layout ,然后选择想要的分屏模式
  • 【6: 】打开sublime text console, View—>Show Console

$\blacksquare$ HTML

  • 【7: 】在html中输入html,p.div等关键词再按Tab键会自动补全
  • 【8: 】打开Command Palette，输入html切换到html编辑
  • 【9: 】Ctrl+p 输入项目中想要浏览的文件，快速定位到
  • 【10: 】Ctrl+/，快速注释
