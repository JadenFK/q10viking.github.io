Windows下Python2,Python3与pip2,pip3

发表于 2018-09-05 | 分类于 python基础

在win8下实验，分别安装python2.7.12,python3.6.4版本，以及pip2,pip3.这些命令能够在机器上同时使用。

配置python环境

  • 将下载好的各个python版本放置在C盘目录下

  • 设置环境变量

    添加路径如： C:\Python27\Scripts\;C:\Python27\;C:\Python3.6.4\Scripts\;C:\Python3.6.4\;

  • 分别在C:\Python27，C:\Python3.6.4下将python.exe改为python2.exe,python3.exe
  • 测试，成功

配置pip2,pip3
