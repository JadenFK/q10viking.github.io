搭建ipython/jupyter notebook服务器

发表于 2018-09-24 | 分类于 机器学习
  • Jupyter notebook(原ipython notebook)是统计，数据分析和机器学习的利器。
  • 一般情况，jupyter notebook网页应用运行在http://localhost:8888用户可以在本地机器上通过浏览器访问和使用。
  • 由于自己的电脑性能不好，将jupyter notebooky应用架设在AWS的服务器上，以获取更强的计算能力和更多的内存。
  • 本文将会以Ubuntu 18.04.1 LTS系统为例，讲解如何安全地搭建jupyter notebook服务器，以允许授权用户使用任意浏览器在任意设备上访问这款应用。

Open a Port

  • To use Jupyter,you need to open port 8888 on your instance’s firefall.
  • Once the port is open you’ll launch the server,then you’ll SSH to your server and create a tunnel for you
$\blacksquare$ Steps
  • Open EC2 dashboard and choose Security Groups on the EC2 navigation bar in the NetWork & Security section.

  • On this page there will be one or more security groups in the list.

  • Find the most recent one(the description has a timesstamp in it)，select it,choose the Inbound tab,and then click Edit.Then click Add Rule. Fill in the fields with following information

Configure a window Client

  • We need to SSH to our server
  • And then create a tunnel for us to use jupyter notebbook
Download PuTTy
  • Download and install PuTTY from the PuTTY download page
Converting Private Key using PuTTYgen
  • For connecting to our EC2 instance using PuTTY,we first need to convert our private key file (.pem) generated by Amazon EC2 to a format that is recognized by PuTTY(.ppk)

$\blacksquare$ Steps

  • Start PuTTYgen(for example,from the Start menu,choose All Programs>PuTTY>PuTTYgen)

  • Under Type of key to generate, choose RSA

  • Choose Load.By default,PuTTYgen displays only files with the extension .ppk.To locate your .pem file,select the option to display files of all types

  • Select your .pem file(‪C:\Users\Viking.ssh\MyPair.pem) for the key pair that you specified when you launched your instance,and then choose Open.Choose OK to dismiss the confirmation dialog box.

  • Choose Save private key to save the key in the format that PuTTY can use.PuTTYgen displays a warning about saving the key without a passphrase. Choose Yes.

  • Specify the same name for the key that you used for the key pair.(for example: MyPair.ppk)
PuTTY Configuration

  • Now open PuTTY and navigate to Session under Category in the left pane.Enter the following information:

  • Now expand Connection on left pane.Navigate to Auth under SSH.Browser and add the .ppk file you created just now.Then click Open at the bottom of screen.

  • If this is the first time you have connected to this instance,PuTTY displays a security alert dialog box that asks whether you trust the host you are connecting to.Choose Yes.A window opens and youi are connected to your instance.

  • Now to set up the SSH tunnel to your EC2 instance,right click on the top left corner of your instance’s window as highlighted below.It will open a dropdown menu.Select Change Settings from the menu to bring up PuTTY Reconfiguration screen.
  • Navigate to Tunnels under SSH on the left pane.Fill up the Source port and Destination as shown below.Select Local and Auto as port forwarding options.Finally click Add followed by Apply to finish setting up the SSH tunnel.

Log into the Jupyter notebook server

  • use the sudo jupyter notebook –allow-root command to start the server.
  • Use Firefox open the URL https://127.0.0.1:8888,and add certification.

  • Fill up the password (created at IPython。See before-Custom SSL and Server Configuration)，such as root
