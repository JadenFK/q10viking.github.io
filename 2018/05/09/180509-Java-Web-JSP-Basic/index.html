JSP基础及入门

发表于 2018-05-09 | 分类于 Servler and JSP

Overview

Web Application

  1. A web site where the HTML pages are generated dynamically
  2. Based on the user’s actions


JSPs and Servlets

  1. Reads user’s actions…normally from HTML Form
  2. Java code that runs on the web server
  3. Performs the work
  4. Returns an HTML page that is generated dynamically

Key Building Blocks

  1. JSP and Servlets are key components of the Java Enterprise Edition(Java EE)
  2. Popular MVC frameworks are actually built on top of JSP and Servlets,such as Spring,JSF,Struts.

First-Example

JSP File

  1. An HTML page with some Java code sprinkled in.
  2. Include dynamic content from Java code

processed

  1. JSP is processed on the server
  2. Result of Java code included in HTML returned to browser

    实体Bean是访问数据库的

  4. JSP+Struts+JavaBeans+DAO
