1) rear,front分别指示当前数组中的最新加入和最早加入的元素。初始化时front=0,rear=capacity-1,size=0以及数组的空间

Insert element at front end
a) first we check deque if Full or not
b) decrement front by 1 and push current key into arrp[front]=key,increment size by 1;

Insert element at rear end
a) first we check deque if Full or not
b) increment rear by 1 and push current key into arrp[rear]=key,increment size by 1;

Delete element at front end
a) First we check deque is empty or not
b) increment front by 1 ,size decrement by 1

Delete element at rear end
a) First we check deque is empty or not
b) decrement rear by 1 ,size decrement by 1
