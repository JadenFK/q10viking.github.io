字典存在的形式

dictionary与defaultdict的区别

The only difference between dictionary and defaultdict is that a defaultdict will have a default value if that key has not been set yet.If you don’t use a defaultdict you’d hava to check to see if that key exists,before you use it.

dictionary如果字典中不存在该键，使用该键会报错

defaultdict默认会有一个值,根据声明的时传入的参数来设定，如果不指定value是什么类型也会报错

setdefault方法

通过使用其方法setDefault，能够避免键不存在时报错

The setdefault() method returns the value of a key (if the key is in dictionary).If not,it inserts key with a default value to the dictionary.

Parameters

  • key - key to be searched in the dictionary
  • default_value(optional) : is inserted to the dictionary if key is not in the dictionary,if not provided,the default_value will be None

使用defaultdict解决下列问题

Problem：you will be given 2 integer,n and m.There are n words,which might repeat,in word group A.There are m words belonging to word group B.For each m words,check whether the word has appeared in group A or not.Print the indices of each occurrence of m in group A.if it dose not appear,print -1.

Sample Input:

5 2
a
a
b
a
b
a
b

Sample Output:

1 2 4
3 5

Algoritm:
a)defaultdict-A以words为键其(indice+1)为值存入字典中（list）
b)将B-group的键值在defaultdict-A中进行查询，如果有值则输出去值，没有则输出-1
