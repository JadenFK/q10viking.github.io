python开发环境搭建

发表于 2018-08-07 | 分类于 python基础

背景

使用Eclipse-Oxygen,在线安装了pyDev6.4.4之后，能够顺利的进行python3开发，但是在开发的过程中，突然发现不能调试，启动调试老是报以下错：

虽然最后找到了一种解决方案，使用python2的解释器就能编译，但是使用python3解释器就不能编译。于是开始怀疑我在线安装的是不是最新版的pyDev。于是卸载掉pyDev插件，使用离线的方法安装，发现Eclipse-Oxygen不能正确显示出python相关的组件，在网上浏览一些信息后，开始怀疑eclipse是不是最新版的问题。… …

果然最后，成功了，且听我慢慢道来。

Eclipse安装pyDev

安装前需要java8和python2.7或python3.4以上的支持

  • 能够进行python3的开发及调试
    目前的日期是：2018-08-07，eclipse在7月份出了最新的eclipse-photon版
    PyDev6.4.4也在七月28号发行。所以接下来下载，安装他们。

在官网上下载eclipse-photon

在PvDev for Eclipse下载PyDev6.4.4

将下载的PyDev 6.4.4.zip和eclipse-jee-photon-R-win32-x86_64.zip分别解压出来。

将PyDev解压的文件中的features文件里面的内容和plugins里的内容分别复制到eclipse解压文件里面的features和plugins里面：

启动eclipse:

配置python解释器：Window->Preferences

打开java的Package Explorer来显示python的项目: windows->show view->other:

创建Python Project: File->new->PyDev Project

创建python文件： 选中项目右键选择：PyDev Module:

更改项目的python解释器

选中项目右键->选择properties

调试

调试代码，设置断点和在Java中一样，双击左边的行号处，然后在代码区域右键Debug as->python run,然后弹出窗口，提示我们要不要切换到调试界面，点击swith进行切换

进行调试：

Note: 简单代码可以用python3调试，但是遇到class,def的这样的代码时还是需要python2.7来进行调试

Eclipse-调试-python快捷键

  • 单步跳入 step into: F5
  • 单步跳过 step over: F6
  • 单步返回 step Return: F7

可以用学校邮箱注册获取免费版，试用期1年，过期再注册即可参考：详细步骤
