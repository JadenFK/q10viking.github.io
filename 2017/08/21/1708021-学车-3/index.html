科目二-II

发表于 2017-08-21 | 分类于 每一天都值得珍惜

练车

在前两天学了左右倒车入库和侧方停车之后，今天直接进入了绕圈的综合练习，包括曲线行驶，直角转弯，坡道停车和起步

直角转弯

  • 在要进行直角转弯的时候，首先得把车速慢下来，将离合器迅速踩到底，然后轻踩刹车，将车速控制住比较慢的速度。
  • 将车调正，用余光将雨刷根部与右边的黄线相切。
  • 当镜子的顶部，（还是车头）与黄线底部相接触的时候，迅速将方向盘往左打死。

曲线行驶

目前让我感到很为难的是，很难把控值车速与打方向盘的关系

  • 教练说看车头，哪边宽往那边打方向盘
  • 向左转弯的时候，看车头与右边线，让车头贴着线走，让线过车头的一半，在差不多要到左边三分之一的时候快速向左打一圈，留着半圈根据情况来修正。
  • 在快要转弯成功的时候，迅速回正，让车身回正过来。
  • 在右转弯的时候，当左边的边线到车中间的时候，迅速往右打一圈，让车头中间贴着线中，剩下的半圈根据情况来修正

坡道停车和起步

  • 在要上坡之前先打左灯，在行驶过程中，让箭头对准自己，视线要望向远方。
  • 在要上坡道后打右灯
  • 在上坡的时候，需要松抬离合，使得有动力上坡，在上坡的时候要让车身正，不要斜着。
  • 当左肩与灯，杆平行的时候，迅速踩下离合，紧接着踩下刹车，然后拉手刹
  • 接着打左灯，然后慢慢松抬离合，当感觉到车有两次抬起的时候，车头微扬，或者汽车发动机抖动的时候，或者发动机发出低沉的声音的时候，松手刹。在松手刹的时候，此时踩着的离合器不要动。当慢慢开走以后再慢慢的完全的松掉离合
  • 在下坡的时候，不仅要踩住刹车，而且还要注意与边线保持距离，调整车头。

注意事项

  • 离合踩要迅速的踩到底，抬起的时候要缓慢，太快乐会导致熄火。慢慢抬离合器，可以完全抬起它。

  • 在操作的过程中要尽量的大胆，只有出问题了，教练才会告诉你问题出在哪里，该怎么去应对。

  • 视线要望向远方，不能老是盯着路线看

科目二第四天训练

对于跑圈，自己基本能够从容的应对了。教练可能也这样觉得，与是在后半部分练习，它带我到测试区测试我的倒车入库，因为倒车入库时接触开车时最先练习的，当时也没什么把握，于是测试很是糟糕，

我发现我的原因老是打方向不够干脆和迅速，这有受我跑圈时候的影响，因为在走曲线的时候总是打圈太快而不能做到游刃有余，所以在左右倒库的时候总是，慢慢的修正，结果总是压线，或者距离边线总是太进。总是在犹豫什么时候打方向盘。

希望明天能够加油了。晚安，世界，晚安，自己。
